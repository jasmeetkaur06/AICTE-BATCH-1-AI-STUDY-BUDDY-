# 📚 AI Study Buddy

An AI-powered learning assistant built for the **EduNet IBM SkillsBuild AICTE Internship**.

## 🚀 Features
- 📖 **Explain Topic** — Explains any topic in simple terms at your level
- 📝 **Summarize Notes** — Summarizes your study notes instantly
- 🧠 **Generate Quiz** — Creates 5 MCQ questions on any topic
- 🃏 **Flashcards** — Generates Q&A flashcards for quick revision

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **AI Model:** LLaMA 3.1 via Groq API
- **Language:** Python
- **Deployment:** Streamlit Cloud

## 💡 How to Run Locally
1. Clone this repo
2. Install dependencies:
   pip install streamlit groq
3. Run the app:
   streamlit run app.py
4. Enter your Groq API key in the sidebar
